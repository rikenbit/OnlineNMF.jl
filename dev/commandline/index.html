OnlineNMF.jl (Command line tool)

All functions can be performed as command line tool in shell window and same options in OnlineNMF.jl (Julia API) are available.

After installation of OnlineNMF.jl, command line tools are saved at YOUR_HOME_DIR/.julia/v1.x/OnlineNMF/bin/.

The functions can be performed as below.

Non-negative Matrix Factorization (NMF)

shell> julia YOUR_HOME_DIR/.julia/v1.x/OnlineNMF/bin/nmf \
--input Data.zst \
--outdir OUTDIR \
--alpha 1 \
--beta 2 \
--graphv 1.0e-8 \
--l1u 1.0e-8 \
--l1v 1.0e-8 \
--l2u 1.0e-8 \
--l2v 1.0e-8 \
--dim 3 \
--numepoch 5 \
--chunksize 1 \
--algorithm frobenius \
--lower 0 \
--upper 1.0f+38 \
--initU U.csv \
--initV V.csv \
--initL L.csv \
--logdir OUTDIR
